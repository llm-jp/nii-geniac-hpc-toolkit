Advanced Deployment Instructions
================================

Terraform group 'primary' was successfully created in directory slurm-a3-base-tky/primary
To deploy, run the following commands:

terraform -chdir=slurm-a3-base-tky/primary init
terraform -chdir=slurm-a3-base-tky/primary validate
terraform -chdir=slurm-a3-base-tky/primary apply

Destroying infrastructure when no longer needed
===============================================

Automated
---------

ghpc destroy slurm-a3-base-tky

Advanced / Manual
-----------------
Infrastructure should be destroyed in reverse order of creation:

terraform -chdir=slurm-a3-base-tky/primary destroy
